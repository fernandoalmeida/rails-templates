# Rails Application Templates

## Overview

A collection of scripts for creating or apply to existing rails applications.

## Usage

    # new apps
    $ rails new blog -m template.rb
    
    # existing apps
    $ rake rails:template LOCATION=template.rb

## Features

### testing_tools.rb

- Guard
- Spork
- Brakeman
- RSpec
- Shoulda
- Factory Girl
- Webrat
